# Sistema de Biblioteca - Java
Este proyecto es una aplicación de consola desarrollada en Java para la gestión básica de una biblioteca, cumpliendo con los requerimientos de la actividad IDS-ActF2.

# Estructura del Proyecto
El sistema cuenta con las siguientes clases principales:

Alumno.java: Define los atributos y métodos para la gestión de usuarios.

Libro.java: Contiene la definición de los libros, sus atributos y estados.

Main.java: Clase principal donde se realiza la instanciación de objetos y la ejecución de los métodos.

# Conceptos de Programación Aplicados
Creación de Clases: Definición de plantillas para objetos Alumno y Libro.

Atributos y Métodos: Implementación de características y comportamientos específicos.

Constructores: Inicialización de objetos con datos específicos.

Instanciación: Creación de objetos reales en la memoria durante la ejecución en la clase Main.

## Instrucciones de Ejecución
1. Descargue o clone este repositorio.
